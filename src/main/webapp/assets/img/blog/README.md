# Blog Images

Thư mục này chứa các ảnh cho trang blog.jsp

## Danh sách ảnh cần:

### Ảnh bìa bài viết:
- `cover.jpg` - Ảnh bìa chính (khuyến nghị: 1200x600px)

### Ảnh minh họa trong bài viết (2 ảnh):
- `illustration-1.jpg` - Hình minh họa 1 (khuyến nghị: 800x600px)
- `illustration-2.jpg` - Hình minh họa 2 (khuyến nghị: 800x600px)

### Ảnh bài viết liên quan (3 ảnh):
- `related-1.jpg` - Bài viết liên quan 1 - "Sự thay đổi của thông tư 200" (khuyến nghị: 800x600px, aspect ratio 4:3)
- `related-2.jpg` - Bài viết liên quan 2 - "Dev tool cùng chúng tôi" (khuyến nghị: 800x600px, aspect ratio 4:3)
- `related-3.jpg` - Bài viết liên quan 3 - "Marketing trong thời đại số" (khuyến nghị: 800x600px, aspect ratio 4:3)
